X-Ray Settings

Default Relays

Events are published to these relays. One WebSocket URL per line (wss://…).

Signing

X-Ray signs events with a NIP-07 browser extension when one is installed. Otherwise it falls back to NSecBunker (remote signer) over WebSocket.

Entities

Publications, people and organizations referenced when building metadata events. Edit as JSON; each is an object keyed by entity id.

People
Organizations

Keypair Registry

X-Ray keeps a registry of entity keypairs it has used to sign. Export a backup, or import one from another device. Private keys are included in the export — handle with care.
